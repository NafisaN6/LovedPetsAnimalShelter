# Loved Pets Animal Shelter

## Overview
A simple static website for an animal adoption shelter. The site displays information about adoptable pets and uses a clean, warm design with custom fonts.

## Project Structure
- `index.html` - Main HTML page
- `style.css` - Styling with custom fonts and responsive images
- `DogCat.png` - Hero image
- `LovedPetsLogo.png` - Site favicon
- `server.py` - Python HTTP server for serving static files

## Technical Setup
- **Type**: Static HTML/CSS website
- **Server**: Python 3 HTTP server (no-cache headers)
- **Port**: 5000 (frontend)
- **Host**: 0.0.0.0 (to work with Replit proxy)

## Recent Changes (Nov 25, 2025)
- Imported from GitHub

## Architecture
Simple static site with no backend or build process. Uses Python's built-in HTTP server to serve files directly.
